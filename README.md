# Flipkart Clone 🛒

A responsive Flipkart-inspired e-commerce website built using **HTML, CSS, and JavaScript**. This project simulates a basic online shopping experience, including user login, product browsing, cart management, and checkout functionality.

## 🚀 Features

* User Login Interface
* Responsive Flipkart-style UI
* Product Catalog Display
* Add to Cart Functionality
* Dynamic Cart Counter
* Checkout Modal Popup
* Delivery Address Form
* Price Calculation
* Order Placement Simulation
* Success Notification Toast

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Google Fonts (Roboto)

1. Clone the repository:

```bash
git clone https://github.com/your-username/flipkart-clone.git
```

2. Open the project folder.

3. Launch `flipkart.html` in any modern web browser.

No additional installation or dependencies are required.

## 📸 Application Flow

### Login Page

* User enters email/mobile number and password.
* Successful login redirects to the store page.

### Store Page

* Displays products in a responsive grid.
* Shows product image, price, discount, and details.
* Users can add products to the cart.

### Checkout

* Clicking **Add to Cart** opens the checkout modal.
* User fills delivery information.
* Order summary and pricing are displayed.
* Clicking **Place Order** completes the purchase simulation.
